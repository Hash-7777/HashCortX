# HashCortx — Brand Guide

## Name
**HashCortx** — Hash (the prompt, the identifier) + Cortex (the thinking layer).
Always written: `HashCortx` — capital H, capital C, no space.
In terminal contexts: `#hashcortx` or `hashcortx_`

## Tagline
> local-first AI coding agents

## Identity Statement (use in all marketing, README, App Store)
```
HashCortx — 100% local. Zero cloud. No account.

Your API keys never leave your keychain.
Your files never leave your machine.
No telemetry. No tracking. No backend server.
Air-gapped capable with local models (Ollama).

Every AI request travels: your device → provider API → back.
Nothing passes through us. There is no "us".
```

## Typography
Font: **JetBrains Mono** (primary), fallback chain in `styles.css`.
Everything is monospace. No sans-serif in the UI.

## Logo Construction
```
# hashcortx_
│ └───────┘└ blinking cursor
└ cyan hash symbol (brand mark)
```
The standalone mark is `#` in `--hc-cyan` with the glow shadow.

## Voice & Tone
- Terminal-first: short, direct, no fluff
- Action-oriented: "patching", "running", "done"
- No marketing speak: never "powerful", "seamless", "revolutionary"
- Security-confident: state facts, not promises

## What HashCortx is NOT
- Not a cloud service
- Not a subscription
- Not a data company
- Not a wrapper around another product
